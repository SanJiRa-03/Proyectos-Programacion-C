# Portafolio de Proyectos y Algoritmia en C

¡Bienvenido/a a mi repositorio de desarrollo en C! Este espacio está dedicado a consolidar y organizar mis soluciones algorítmicas, prácticas académicas y proyectos de ingeniería. El objetivo principal de este portafolio es asentar bases sólidas de lógica de bajo nivel, gestión de memoria, modularidad y optimización de código.

## 📂 Estructura del Repositorio

El repositorio está organizado por módulos independientes. Cada carpeta contiene el código fuente correspondiente junto con su propia documentación detallada:

*   **[01-Gestion-Alumnos-Structs](./Proyectos-Programacion-C/01-Gestion-Alumnos-Structs):** Sistema de gestión académica que implementa estructuras (`struct`), validación estricta de datos por consola y modularización.
*   **[02-Agente-Autonomo-Manhattan](./Proyectos-Programacion-C/02-Agente-Autonomo-Manhattan):** Simulación bidimensional de un agente robot que toma decisiones de ruta óptima utilizando la Distancia de Manhattan.
*   **[03-Gestion-Deportistas-Ficheros](./Proyectos-Programacion-C/03-Gestion-Deportistas-Ficheros):** Sistema interactivo que utiliza arrays de estructuras para administrar perfiles deportivos, incluyendo lógica de reorganización dinámica de memoria tras la eliminación de registros y persistencia de datos.

## 🛠️ Tecnologías y Conceptos Clave

A lo largo de los diferentes proyectos se aplican los pilares fundamentales del lenguaje C y la ingeniería de software:

- **Estructuras de Datos:** Uso de variables primitivas, arrays bidimensionales y estructuras personalizadas (`struct`).
- **Lógica Algorítmica:** Optimización de flujos iterativos (`do-while`, `for`), condicionales complejos, algoritmos de desplazamiento de elementos para baja de registros y control de entornos pseudoaleatorios.
- **Modularidad:** División de responsabilidades mediante la creación de funciones y paso de parámetros.
- **Persistencia de Datos:** Manipulación, flujo de escritura (`fopen`, `fprintf`) y actualización limpia de ficheros de texto (`.txt`) para asegurar la consistencia del estado de la aplicación.
